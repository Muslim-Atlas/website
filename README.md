# Muslim Atlas — Web Landing Page

The official, production-grade landing page for **Muslim Atlas**, a comprehensive mobile companion application designed to connect users with mosques, halal dining options, and highly-accurate prayer times.

Live Site: [muslimatlas.app](https://muslimatlas.app)

---

## Features

- **Product Showcase**: Detailed overview of app features (Mosque Finder, Halal Food directory, Adhan calculations, and Android Widget).
- **Download Hub**: Quick links to download the latest signed Android APK directly from GitHub releases or external stores.
- **Interactive Guides**: User documentation explaining high-latitude adjustment rules, calculation methods (London Unified, ISNA, etc.), and widget customization.
- **Responsive Layout**: Designed from scratch with custom styling, optimized for both desktop and mobile viewports.

| Component | Technology |

---

## Getting Started

### 1. Installation
```bash
npm install
```

### 2. Environment Setup
Create a `.env.local` file in the project root:

```ini
# Site Configuration
NEXT_PUBLIC_SITE_URL=https://muslimatlas.app
```

### 3. Running the Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
npm run start
```
